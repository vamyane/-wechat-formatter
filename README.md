# 公众号排版器 Pro v2

一站式微信公众号文章排版工具，支持 **82 个精选模板**，覆盖 6 大风格分类。

## ✨ 功能特色

- 🎨 **82 个模板** — 浅色/深色/渐变/边框/色块/特殊风格全覆盖
- 📂 **智能分类** — 每个分类标注推荐内容类型，快速找到合适模板
- 📱 **实时预览** — 手机壳模拟，所见即所得
- 📋 **一键复制** — 复制后直接粘贴到公众号编辑器
- 🖼️ **图片支持** — 拖拽/粘贴图片，自动适配模板风格
- 📐 **LaTeX 公式** — 支持数学公式渲染
- 🎯 **快速插入** — 结尾引导、作者简介、提示框等常用组件
- 🎨 **自定义配色** — 主色调、背景色、字色自由调节

## 📂 模板分类

| 分类 | 数量 | 适合内容 |
|------|------|----------|
| ☀️ 浅色系 | ~20 | 通用、表格、数据、正式文档 |
| 🌙 深色系 | ~12 | 科技、潮流、影视、品牌展示 |
| 🌈 渐变风 | ~10 | 创意、营销、情感、旅行 |
| 🖼️ 边框装饰 | ~12 | 公告、文艺、产品介绍 |
| 🎨 色块强调 | ~10 | 新闻、观点、活动推广 |
| ✨ 特殊风格 | ~18 | 复古、禅意、科幻、艺术等 |

## 🚀 使用方法

1. 双击打开 `index.html`
2. 在左侧编辑区粘贴或输入文章内容
3. 点击「模板库」选择喜欢的模板和分类
4. 右侧实时预览效果
5. 点击「复制到公众号」，粘贴到公众号编辑器即可

## 🔧 技术栈

- 纯 HTML/CSS/JS，无需构建工具
- KaTeX 数学公式渲染
